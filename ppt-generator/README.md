# PPT Generator

A PowerPoint presentation generator application that creates professional presentations from documents using AI.

## Project Structure

- `backend/` - Flask backend service for processing and generating presentations
- `frontend/` - React frontend application

## Setup

### Backend
```bash
cd backend
pip install -r requirements.txt
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Features

- Upload documents and generate presentations
- Multiple template styles (Professional, Modern, Financial, Executive)
- AI-powered slide generation
- Preview and download PowerPoint files
